[마지막 해결책: 웹 메일(OWA)을 통한 설치]
회사 보안 정책으로 PC 레지스트리/메뉴가 막혀있을 확률이 99%입니다.
이 경우, PC 설정과 무관하게 "웹 메일"에서 설치하면 데스크탑으로 동기화시키는 우회 방법을 써야 합니다.

1. 방화벽 확인
   - 크롬이나 엣지 브라우저를 켜고 아래 주소에 접속해 보세요.
   - https://sepu1329-blip.github.io/outlook-ai-assistant/index.html
   - **빈 화면**이나 **글자**가 나오면 성공. "차단됨(Blocked)"이 뜨면 회사에서 외부 사이트를 막은 것입니다 (이 경우 사용 불가).

2. 웹 메일 접속
   - 회사의 웹 메일 주소(예: https://outlook.office.com 또는 사내 OWA 주소)로 접속해서 로그인하세요.

3. 웹에서 설치
   - 아무 **새 편지 쓰기** 창을 엽니다.
   - 메시지 창 하단 툴바의 **[...]** 또는 **[앱]** 아이콘 클릭.
   - **[추가 기능 가져오기 (Get Add-ins)]** 클릭.
   - **[내 추가 기능 (My Add-ins)]** 탭 선택.
   - **[사용자 지정 추가 기능 (Custom Add-ins)]** 섹션 찾기.
   - **[파일에서 추가...]** 클릭 후 `release` 폴더의 `manifest.xml` 선택.

4. 확인
   - 웹 메일에서 설치가 성공하면, PC 아웃룩을 껐다 켜세요.
   - 잠시 후 PC 아웃룩에도 자동으로 아이콘이 생깁니다.
